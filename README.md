# Dotfiles

1. Download iTerm2 and font-hack from brew
```bash
brew install iterm2
brew install font-hack
```

2. Downlaod repo and install
```bash
git clone repo-url
./path/to/repo/install.sh
```

3. Load iTerm2 preferences from dotfiles/iTerm2 folder

4. Install vim plugins
```vim
:PluginInstall
```
